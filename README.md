# Notebook

## Installation

1.  开发系统基于 macOS NodeJS v10.x
2.  运行前 需要在 client、server 文件夹下运行安装依赖

        cd client/ && yarn install

        cd server/ && yarn install

## Documentation

1.  Client

    项目基于 React

    运行命令:

        cd client/ && yarn start

    运行成功可访问 http://localhost:3000

2.  Server

    项目基于 Express

    运行命令:

        cd server/ && yarn start
